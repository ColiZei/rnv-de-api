# REST API RNV MAP

## Install:

```
npm install
```

## Start in Dev mode:

```
npm run dev
```

# Endpoint examples:

- Get all Locations:

```
GET /locations/
```

- Get Location:

```
GET /location/2
```

- Add Location:

```
POST /location/
{
  name: "I am a Location!"
}
```

- Update Location:

```
PUT /location/1
{
  name: "Change my name!"
}
```

- Update Location:

```
DELETE /location/1
```
